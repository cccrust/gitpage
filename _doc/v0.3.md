# v0.3 — Pages 功能

## 新增功能

- **Pages 部署引擎**: `deploy_pages()` 將 bare repo 中指定 branch+source_dir 的內容提取到 `/pages/` 目錄
- **啟用/設定 Pages**: 透過 UI 或 API 設定 branch、source_dir、custom_domain
- **自動部署**: 啟用 Pages 時自動部署；git push 後自動重新部署
- **手動重新部署**: `POST /api/pages/:repo_id/deploy`
- **Pages 設定頁面**: 前端 `/repo/:id/pages` 設定頁面
- **Serving**: `/pages/{username}/{repo}/{*path}` 提供靜態網站服務

## API 變更

| 方法 | 路徑 | 說明 |
|------|------|------|
| POST | /api/pages/:repo_id/deploy | 手動重新部署 Pages |

PUT /api/pages/:repo_id 啟用時會自動部署。

## 測試
