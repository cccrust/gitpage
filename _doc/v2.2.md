# v2.2 — Star / Watch / Fork 社交機制

類似 GitHub 的 star（書籤/指標）、watch（通知訂閱）、fork 計數。

## 動機

- Star 作為使用者對專案的關注指標（書籤/讚）
- Watch 讓使用者訂閱專案更新通知
- Fork 計數反映專案的衍生使用情況
- 讓 gitpage 具備基礎社交互動能力

## 實作細節

### 資料表

```sql
CREATE TABLE stars (
    user_id    INTEGER NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    repo_id    INTEGER NOT NULL REFERENCES repositories(id) ON DELETE CASCADE,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    PRIMARY KEY (user_id, repo_id)
);

CREATE TABLE watches (
    user_id    INTEGER NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    repo_id    INTEGER NOT NULL REFERENCES repositories(id) ON DELETE CASCADE,
    watch_type TEXT NOT NULL DEFAULT 'participating',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    PRIMARY KEY (user_id, repo_id)
);
```

### Repositories 新增欄位

```sql
ALTER TABLE repositories ADD COLUMN stars_count INTEGER NOT NULL DEFAULT 0;
ALTER TABLE repositories ADD COLUMN forks_count INTEGER NOT NULL DEFAULT 0;
ALTER TABLE repositories ADD COLUMN watch_count INTEGER NOT NULL DEFAULT 0;
```

計數透過 `SELECT COUNT(*)` 子查詢即時更新（INSERT/DELETE 後自動重算）。

### 資料模型

`Repository` struct 新增三個欄位：

```rust
pub struct Repository {
    // ...既有欄位...
    pub stars_count: i64,
    pub forks_count: i64,
    pub watch_count: i64,
}
```

此欄位已加入所有 DB SELECT 查詢（7 處 `map_repo_row` 調用 + `get_all_ssh_keys` + `list_org_repos_with_orgname`）及 `create_repo` 建構子。

### DB 方法

| 方法 | 說明 |
|------|------|
| `star_repo(user_id, repo_id)` | INSERT INTO stars + UPDATE stars_count |
| `unstar_repo(user_id, repo_id)` | DELETE FROM stars + UPDATE stars_count |
| `is_starred(user_id, repo_id)` | SELECT COUNT(*) |
| `list_stargazers(repo_id)` | JOIN users ON stars |
| `list_user_stars(user_id)` | JOIN repositories ON stars |
| `watch_repo(user_id, repo_id, type)` | INSERT OR REPLACE + UPDATE watch_count |
| `unwatch_repo(user_id, repo_id)` | DELETE + UPDATE watch_count |
| `get_watch_type(user_id, repo_id)` | SELECT watch_type |

### 前端變更

**RepoPage**：
- 顯示 ★ Star N、🍴 N、👁 N 計數
- Star 按鈕可點擊切換（★ / ☆），即時更新計數

**Dashboard**：
- 新增 Repos / Starred 兩個分頁
- Starred 分頁顯示使用者 star 過的倉庫列表
