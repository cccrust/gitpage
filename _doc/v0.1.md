# Gitpage v0.1 — 核心後端系統

## 功能

### REST API
- 使用者註冊、登入、個人資料查詢
- 倉庫 CRUD（建立、列表、查詢、刪除）
- 公開倉庫瀏覽
- 目錄與檔案瀏覽（支援子目錄）
- Markdown 檔案渲染（透過 pulldown-cmark）
- README 自動偵測與渲染
- Git 提交紀錄查詢

### Git HTTP Smart Protocol
- 完整支援 git clone / fetch / push
- 透過 `git http-backend` 實作
- 倉庫路徑：`/git/{username}/{repo_name}`

### Pages（基礎架構）
- Pages 設定 CRUD API
- 靜態檔案服務路由準備就緒

## 技術棧

- **後端框架**: Rust + Axum 0.7.9
- **資料庫**: SQLite (rusqlite)
- **Git**: git2 + git http-backend
- **認證**: JWT (jsonwebtoken) + Argon2 密碼雜湊
- **Markdown**: pulldown-cmark
- **路由**: Axum Router 使用 `:param` 語法

## API 路由

| 方法 | 路徑 | 說明 | 認證 |
|------|------|------|------|
| POST | `/api/auth/register` | 註冊 | 否 |
| POST | `/api/auth/login` | 登入 | 否 |
| GET | `/api/auth/me` | 目前使用者 | 是 |
| GET | `/api/repos` | 列出我的倉庫 | 是 |
| POST | `/api/repos` | 建立倉庫 | 是 |
| GET | `/api/repos/:id` | 倉庫資訊 | 否 |
| DELETE | `/api/repos/:id` | 刪除倉庫 | 是 |
| GET | `/api/users/:username/repos` | 使用者公開倉庫 | 否 |
| GET | `/api/:user/:repo/tree` | 目錄內容 | 否 |
| GET | `/api/:user/:repo/blob` | 檔案內容 | 否 |
| GET | `/api/:user/:repo/readme` | README | 否 |
| GET | `/api/:user/:repo/commits/:branch` | 提交紀錄 | 否 |
| GET/POST | `/git/:user/:repo/*` | Git Smart Protocol | 否 |
| GET | `/pages/:user/:repo/*` | Pages 靜態檔案 | 否 |

## 目錄結構

```
src/
├── main.rs          # 伺服器入口
├── app.rs           # Axum 路由與中介軟體
├── config.rs        # 設定檔
├── auth/mod.rs      # JWT 發行與驗證
├── db/
│   ├── mod.rs       # SQLite CRUD
│   └── models.rs    # 資料結構
├── git/mod.rs       # Git 操作 + HTTP Backend
├── handlers/
│   ├── mod.rs
│   ├── auth.rs      # 認證處理
│   ├── repos.rs     # 倉庫處理
│   ├── content.rs   # 檔案瀏覽處理
│   ├── pages.rs     # Pages 設定處理
│   └── git_smart.rs # Git HTTP 處理
└── utils/errors.rs  # 錯誤類型
static/              # 前端靜態資源
config.toml          # 伺服器設定
test.sh              # 完整整合測試
```
