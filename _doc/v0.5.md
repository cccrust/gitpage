# v0.5 — Pages 功能完成 + UI 強化

## 新增功能

### Pages 靜態網站託管
- `GET /api/pages/:repo_id` — 取得 Pages 設定
- `PUT /api/pages/:repo_id` — 新增/更新 Pages 設定（啟用時自動部署）
- `POST /api/pages/:repo_id/deploy` — 手動觸發部署
- `deploy_pages()` 從 git tree 遞迴解壓檔案到 `<storage>/<user>/<repo>/pages/`
- `serve_pages()` 從 pages 目錄提供靜態檔案，目錄自動尋找 index.html
- 自動部署：git push 成功後背景執行 `auto_deploy_pages()`
- 前端 `/repo/:id/pages` 設定頁面（branch、source_dir、custom_domain、enabled、Redeploy 按鈕）

### 表單美工強化
- 全域 input / textarea / select 統一樣式，focus box-shadow 動畫
- `.btn`（黑色實心）、`.btn-sm`（邊框）、`.btn-sm.danger`（紅色）
- checkbox 使用 `accent-color` 原生美化
- `.danger-zone` 紅色邊框卡片區域（刪除確認）
- `.msg-ok`（綠色）/ `.msg-err`（紅色）/ `.hint`（灰色說明）

### 演示頁面
- alice/blog 新增 `pages/` 目錄，包含 index.html / style.css / app.js / about.html
- 展示 Pages 功能：部落格首頁 + 關於頁面 + CSS 樣式 + JS 互動

## Bug 修復
