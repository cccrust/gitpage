# v0.4 — 強化功能

## 新增功能

### 私有倉庫存取控制
- `resolve_repo()` helper 在 content handlers 中統一檢查
- 私有倉庫要求認證 token 且必須是倉庫擁有者
- 未認證或非擁有者回傳 401 UNAUTHORIZED

### 使用者個人檔案
- `GET /api/users/:username/profile` — 公開個人檔案 + 公開倉庫列表
- `PUT /api/users/:username/profile` — 更新 bio/avatar_url (需認證)
- 前端 `/u/:username` 個人檔案頁面

### 倉庫設定
- `PUT /api/repos/:id` — 更新 description、is_private
- 前端 `/repo/:id/settings` 設定頁面
- 危險區域：輸入倉庫名稱確認後可刪除

### 搜尋
- `GET /api/repos/search?q=xxx` — 搜尋公開倉庫 (name/description LIKE)
- 前端 Dashboard 搜尋列 (Enter 觸發)

### UI 強化
- 底部導航移至頂部導航
- Repo 頁面包屑顯示使用者名稱
- Dashboard 連結修正為 `/repo/:id`

## API 變更

| 方法 | 路徑 | 說明 |
|------|------|------|
| GET | /api/users/:username/profile | 使用者個人檔案 + 公開倉庫 |
| PUT | /api/users/:username/profile | 更新個人檔案 |
| PUT | /api/repos/:id | 更新倉庫設定 |
| GET | /api/repos/search?q= | 搜尋公開倉庫 |
