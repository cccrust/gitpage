# v0.2 — React 前端

## 新增功能

- **React + Vite + TypeScript** 前端 SPA
- **認證頁面**: 登入 (LoginPage) / 註冊 (RegisterPage)
- **儀表板 (Dashboard)**: 列出我的倉庫、建立新倉庫
- **倉庫頁面 (RepoPage)**: 顯示檔案列表、README、最新 commits
- **檔案瀏覽 (FileViewPage)**: 目錄/檔案瀏覽，Markdown 渲染
- **提交歷史 (CommitsPage)**: 列出所有 commits
- **mobile-first CSS**: Threads/X.com 極簡風格 (黑/白/灰)
- **API 客戶端 (api.ts)**: typed fetch wrapper，自動帶 JWT
- **Layout 元件**: 頂部導航 + 底部標籤欄
- **MarkdownView 元件**: 渲染 pulldown-cmark 輸出的 HTML

## 後端變更

- **app.rs**: fallback handler 支援 `frontend/dist/` + SPA fallback (serve index.html for non-file paths)
- 無 API 變更，向前相容

## 啟動方式

```bash
# Build frontend
cd frontend && npm run build

# Run backend (serves frontend from frontend/dist/)
cd .. && cargo run

# Or dev mode with proxy
cd frontend && npm run dev
# (proxies /api /git /pages to localhost:8080)
```
