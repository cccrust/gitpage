# v2.0 — Issue 與 Pull Request 基礎協作

Issue/PR 系統 + Fork 功能，讓 gitpage 具備基礎協作能力。

## 動機

- 從單純的 Git 託管 + Pages/App 部署，邁向協作平台
- Issue 管理問題與功能請求，PR 實作程式碼審查流程
- Fork 作為協作起點（fork → branch → PR → merge）
- process 與 docker 兩種模式皆可使用

## 實作細節

### Issue Number / PR Number 自動遞增

```sql
SELECT COALESCE(MAX(number), 0) + 1 FROM issues WHERE repo_id = ?
```

各自獨立序列（issue 和 PR 各有自己的 number，互不衝突）。

### 資料表

見 `src/db/mod.rs` migration section（version >= 20250601）：

- `issues` — repo_id, number, title, body, state, author_id, assignee_id, timestamps
- `issue_labels` — repo_id, name, color
- `issue_label_map` — issue_id, label_id (M2M)
- `issue_comments` — issue_id, author_id, body, timestamps
- `pull_requests` — repo_id, number, title, body, state, author_id, head_repo_id, head_ref, base_ref, merge_commit_sha, timestamps
- `repositories.forked_from` — ALTER TABLE 新增

### Fork 實作

使用系統 `git clone --bare`（因 `git2::RepoBuilder` 不是 `Send`，無法在 async context 使用）：

```rust
Command::new("git")
    .args(["clone", "--bare", &source_path, &new_path])
    .output()?;
```

其餘所有 git 操作皆使用 libgit2（git2 crate）。

### Merge 實作（libgit2 3-way merge）

```rust
let merge_base = repo.merge_base(base_oid, head_oid)?;
let merge_tree = repo.merge_trees(&ancestor_tree, &head_tree, &base_tree, None)?;
let index = merge_tree.index();
if index.has_conflicts() { return conflict error; }
let tree_oid = index.write_tree_to(&repo)?;
let merge_commit = repo.commit(Some("refs/heads/main"), ...)?;
```

- 有 conflict 時拒絕 merge，回傳 conflict file list
- 成功後更新 base ref 到 merge commit，記錄 `merge_commit_sha`
- PR state 改為 "merged"，記錄 `merged_at`

### 前端

| 頁面 | 路徑 | 說明 |
|------|------|------|
| IssueList | `/repo/:id/issues` | 標籤切換 open/closed，列表顯示 title + number + state badge |
| IssueNew | `/repo/:id/issues/new` | Title + Body 編輯器 |
| IssueDetail | `/repo/:id/issues/:issueNumber` | Issue 主體 + 留言串 + close/reopen/delete 按鈕 |
| PRList | `/repo/:id/pulls` | 標籤切換 open/closed/merged |
| PRNew | `/repo/:id/pulls/new` | Head repo 選擇 + branch + title/body |
| PRDetail | `/repo/:id/pulls/:prNumber` | PR 資訊 + Diff 列表 + Merge 按鈕 |
| RepoPage | `/repo/:id` | Issues / PRs 導航分頁 + Fork 按鈕 |
