# v1.0 — Gitpage 穩定版

輕量級、自架 Git 程式碼託管平台。類似最小化的 GitHub/GitLab，適合個人或小團隊。

---

## 功能總覽

| 功能 | 說明 |
|------|------|
| Git 託管 | HTTP Smart Protocol push/pull/clone，libgit2 讀取 tree/blob/commit |
| Pages 靜態網站 | 自動部署，push 即觸發 |
| App Hosting | 直接 subprocess 啟動 web 專案，每個 app 獨立 port |
| 檔案管理 | Dropbox 風格 staging 目錄，batch commit 到 git |
| 部署日誌 | 每次 deploy 記錄輸出與狀態 |
| SSH Shell | OpenSSH 整合，`ssh` 進 staging 目錄操作 |
| Repo 管理 | 建立、刪除、重新命名、公開/隱藏設定 |
| 搜尋 | 公開 repo 全文搜尋 + 分頁 |
| 使用者設定 | 修改密碼、個人簡介、頭像 |
| JWT 認證 | argon2 密碼 + jsonwebtoken，支援環境變數覆蓋 secret |
| 錯誤中文化 | 所有後端錯誤訊息顯示中文 |
| 響應式 UI | React 19 + TypeScript + Vite，行動裝置適用 |

---

## 設定

```toml
[server]
host = "0.0.0.0"
port = 8080

[database]
path = "data/gitpage.db"

[jwt]
secret = "change-me-in-production"          # 或設 JWT_SECRET 環境變數
expires_in_hours = 24

[ssh]
enabled = true

[cors]
allowed_origins = ["*"]

[upload]
max_file_size = 10485760

[apps]
port_range_start = 4000
port_range_end = 65535
```

## 目錄結構

```
data/
├── gitpage.db              SQLite
├── repos/{u}/{r}.git       裸倉庫
├── staging/{u}/{r}/        檔案管理暫存區
└── apps/{u}/{r}/           App 建置工作區

src/
├── main.rs                 入口：設定、DB、SSH、啟動
├── app.rs                  路由 + fallback handler
├── config.rs               設定結構
├── auth/mod.rs             JWT 建立/驗證（OnceLock 全域 secret）
├── db/
│   ├── mod.rs              SQLite 操作 + 遷移
│   └── models.rs           資料結構
├── git/mod.rs              libgit2 + git http-backend
├── handlers/               各功能 handler
│   ├── auth.rs             註冊/登入/密碼/個人資料
│   ├── repos.rs            CRUD/搜尋/重新命名
│   ├── content.rs          tree/blob/readme/commits
│   ├── files.rs            檔案管理
│   ├── pages.rs            Pages 設定/部署
│   ├── apps.rs             Apps 設定/部署/日誌
│   ├── ssh_keys.rs         SSH 金鑰管理
│   └── git_smart.rs        Pages 檔案伺服
├── deploy.rs               App 子程序管理
└── ssh.rs                  authorized_keys 產生
```

## 啟動方式

```bash
# 開發模式
cargo run                            # 後端 :8080
cd frontend && npm run dev           # 前端 :5173（HMR，代理 /api /git /pages）

# 正式執行
./run.sh                             # 建置前端 + release 後端 → :8080

# 測試資料
./seed.sh                            # 建立 alice/bob 帳號及範例 repo
./test.sh                            # 整合測試（清空 data/ 重新開始）
```
